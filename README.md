# Band Name Generator

Put words in a comma-separated text file named like this.

```
rolling
stone, stones
flaming
lip, lips
```

Words on same line won't be combo'd.

Run like this.

```
./band_name_generator.py example_words_file.txt
```

The output of above example is this.
